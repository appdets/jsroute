Welcome to Vanilla Router!
Vanilla Router is a simple web router kit developed with Vanilla JavaScript only. Basically, it's developed targeting WordPress admin panel screen for Plugins/Themes developer. Unlike React or Vue router, Vanilla Router works with existing DOM. No need to have components in Virtual DOM.

Install using NPM
npm i vanillarouter
